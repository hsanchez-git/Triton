# Triton

Repositorio dedicado al aprendizaje, experimentación y desarrollo de proyectos de programación GPU con Triton.

## Contenido

### Curso

La carpeta `curso` contiene el aprendizaje progresivo de Triton, desde la instalación del entorno hasta el desarrollo de kernels avanzados.

### Proyectos

La carpeta `proyectos` contiene aplicaciones y proyectos completos desarrollados con Triton.

### Benchmarks

La carpeta `benchmarks` contiene pruebas y comparaciones de rendimiento entre kernels, configuraciones y dispositivos.

### Documentación

La carpeta `documentacion` contiene apuntes generales, referencias y explicaciones técnicas.

## Estructura del repositorio

    Triton/
    ├── curso/
    │   └── 00_entorno/
    ├── proyectos/
    ├── benchmarks/
    ├── documentacion/
    └── scripts/

## Objetivos

- Comprender la arquitectura básica de las GPU.
- Aprender el modelo de programación de Triton.
- Implementar kernels personalizados.
- Comprobar la corrección de los resultados.
- Medir y optimizar el rendimiento.
- Integrar Triton con PyTorch.
- Desarrollar proyectos completos de programación GPU.
